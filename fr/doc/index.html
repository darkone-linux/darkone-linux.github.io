Le Projet

Non informaticien ?

Le guide de l’utilisateur est pour vous !

Une configuration NixOS multi-utilisateur, multi-hôte et multi-service :

  • 🔥 Déclaratif, reproductible, immuable 🡕.
  • 🚀 Modules prêts à l’emploi.
  • ❄️ Configuration 🡕 simple.
  • 🧩 Organisation cohérente.
  • 🌎 Un réseau complet.

Note

Ce projet évolue en fonction de mes besoins. Si vous souhaitez être informé des prochaines versions stables, merci de me le faire savoir sur GitHub 🡕 ou en vous abonnant à ma chaîne YouTube 🡕 (FR). Merci !

Couches d’abstraction

Section intitulée « Couches d’abstraction »

Les niveaux inférieurs rendent service aux niveaux supérieurs.

Ces configurations sont organisées par catégories:

  • Modules mixin qui composent et configurent les hôtes.
  • Modules de service pour les services auto-hébergés.
  • Modules système de configuration système de base.
  • Modules de sécurité pour le durcissement du système.
  • Applications CLI outils en ligne de commande.
  • Applications GUI applications graphiques.
  • Modules d’administration pour la gestion du parc.
  • Modules de gestion d’utilisateurs pour les comptes et accès.
  • Modules Home Manager pour les profils utilisateur.
  • Profils utilisateur, configurations Home Manager 🡕 standards.
  • Outils pour maintenir une configuration qualitative.

État des services SSO (OIDC / Oauth2)

Section intitulée « État des services SSO (OIDC / Oauth2) »
  • Oauth2 = permet une connexion oauth2 / oidc
  • Natif = pas besoin de plugin ou autre, on peut paramétrer directement
  • PKCE = prend en charge PKCE
  • Décl. (déclaratif) = tous les paramètres peuvent être déclarés dans la configuration
  • OK = implémentation fonctionnelle
  • ☑️ = appliqué par un script / démon idempotent (pas du pur déclaratif)

Utilisation de l’IA

Section intitulée « Utilisation de l’IA »

Introduction de l’IA

Jusqu’à fin avril 2026, le code de ce projet a été entièrement écrit à la main, sans aide de l’IA. Parce que l’assistance IA va devenir indispensable pour s’adapter à un nouveau rythme de développement global, une politique est en cours de mise en place depuis mai 2026, de préférence souveraine et locale.
